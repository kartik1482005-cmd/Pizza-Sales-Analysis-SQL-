# Pizza Sales Analysis (SQL)

## Overview
This project analyzes pizza sales data using SQL to extract meaningful business insights.

## Description
Performed data analysis on pizza sales dataset using SQL (joins, aggregations, GROUP BY, HAVING) to identify revenue trends and top-selling products.

## Skills Used
- Joins (INNER, LEFT)
- Aggregations (SUM, COUNT)
- GROUP BY, HAVING
- Filtering

## Key Insights
- Top selling pizzas
- Revenue trends
- Customer order patterns

## Tools
- MySQL / SQL Server
